Dota Motivator подключается к Dota 2 через Game State Integration, считает каждый матч, ведёт виртуальный баланс и не даёт зайти в катку, когда уже пора отдохнуть.

Скачать для Windows GitHub

Windows 10/11 · MacOS и Linux — в планах

Автоматический учёт

GSI ловит начало и конец матча, без ручного ввода. Видишь всю историю своих игр.

Виртуальный баланс

Задай стартовый баланс и долговой порог. Каждая игра меняет состояние — наглядно и честно.

Блокировка катки

Перешёл долговой порог — игра не начнётся. Меньше тильта, больше дисциплины.

Как это работает

  1. 1

    Авторизация через Steam

    Один клик — подключаешь свой Steam-аккаунт.

  2. 2

    Установка GSI-конфига

    Приложение само находит установку Dota 2 и кладёт нужный конфиг.

  3. 3

    Играй

    Каждый матч автоматически попадает в историю, баланс пересчитывается.
